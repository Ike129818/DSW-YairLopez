Proposito del Sitio

El sitio web tiene como fin abordar uno de los temas principales para el aprendizaje de HTML5, por cual se busca de una manera sencilla explicar le funcionamiento de los sitios web mediante el uso de la herramienta. La tematica que se abordará inicialmente es el uso de las etiquetas básicas HTML5. Posterior a ello se relacionanran conceptos concernientes a los atributos y valores empleados enlos diferentes sitios web que pueden ser creador mediante esta estructura básica y código.

Etiquetas Básicas HTML5

Para iniciar debemos conocer lo que es una etiqueta en HTML5, según Xavier Idevik (2021) Una etiqueta es la que le indica al navegador donde inicia y donde finaliza cada elemento de los que está compuesto.

Las etiquetas se componen de tres segmentos, así:

  • Etiquetas de Apertura: Se utilizan para indicar donde empieza un elemento. Se encuentra dentro de corchetes de apertura y cierre.
  • Contenido: Es el contenido que la audiencia del sitio web puede visualizar en sus diferentes maneras como lo son: texto, imagen y sonido.
  • Etiquetas de Cierre: Se comportan de la misma manera que las etiquetas de apertura pero al cerrar la etiqueta de debe anteponer una barra inclinada delante del nombre del elemento.

    • Algunas de las etiquetas más empleadas para la creación de cualquier docuemnto HTML son las siguientes:

    < body > < /body >

    Indica la parte de cuerpo del contenido de un documento HTML. Es una etiqueta esencial para cualquier documento ya que nos indica donde inicia el contenido visible.

    < head > < /head >

    Es la parte superior del documento HTML, es donde podremos indicar los metadatos: título del documento, hojas de estilos, javascript, CSS

    < div > < /div >

    Un elemento que es usado mayoritariamente para agrupar otros elementos y actuar como plantilla de otros controles. Esta etiqueta nos ayuda a estructurar el documento en secciones

    < a > < /a >

    Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href, donde pondremos el enlace al que queremos conectar. Otro atributo muy usado es target, el cual nos sirve para indicar si el enlace se abrirá en una nueva ventana o en la misma.

    < strong > < /strong >

    Si el documento contine mucho texto, es importante poder dar énfasis a una parte en concreto, lo que la etiqueta en mención nos permite hacer.

    < br >

    Con esta etiqueta le indicamos al navegador que viene un salto de línea. Permitiendo así un texto más legible, se debe ubicar al final del texto donde se quiere dar el salto de línea

    < IMG > < /IMG >

    Usamos la etiqueta IMG para mostrar imágenes dentro del contenido. Necesita el atributo src para funcionar, ya que será donde indicaremos desde donde tiene que mostrar la imágen

    < OL > < li > < /li > < /OL > | < UL > < li > < /li > < /UL >

    Las etiquetas OL y LI nos sirven para crear listas, OL para listas ordenadas y UL sin orden. Dentro de las listas, los elementos se identifican con la etiqueta li.

    < p > < /p >

    Etiqueta que nos sirve para agrupar texto dentro de un párrafo. Su finalidad es permitir hace el contenido más organizado y fácil de leer

    < SPAN > < /SPAN >

    Esta etiqueta nos permite personalizar el estilo de tan solo una parte del texto

    Atributos y Valores

    Los atributos pueden considerarse como información adicional que se coloca siempre en la etiqueta de apertura, esto para añadir, complementar información o dar significado al elemento HTML que estamos colocando. Los atributos HTML van siempre dentro de la etiqueta de apertura y sirven para modificar o agregar alguna característica al contenido

    Atributos más usados en HTML

  • Id = "". Para identificar un elemento único.
  • class = "". Identificador múltiple.
  • align = "". Alineación de contenido.
  • border = "". Para darle borde al contenido.
  • style = "". Para darle un estilo al contenido.
  • background-color = "". Para color de fondo.
  • href = "". Para enlaces html.
  • height = "". Para determinar altura.
  • width = "". Para determinar ancho
  • src = "". Para imagenes

    • Valores de un atributo

    Cuando se hace uso de un atributo html se debe especificar su valor, este se puede representar mediante una cantidad exacta, característica especifica o tan solo en que medida se ejecutara dicho atributo. El valor se expresa entre comillas después del signo de (=), de acuerdo al atributo, pueden existir una variedad de valores disponibles.

    A continuación se relacionan algunos atributos y valores que pueden tomar

  • Id. Su valor puede ser cualquier palabra que sirva para identificar un elemento.
  • align. alineación, sus valores pueden ser: "center, right, left, justify".
  • border-style. estilo de borde, los valores a tomar son: "solid, dotted, dashed, double, etc".
  • bgcolor.Para color de fondo, para ciertos elementos como body puede ser cualquier color en formato palabra, hexadecimal, RGB.
  • href. Para enelaces s valor puede ser la direción de una carpeta o URL.
  • heigh. para definir altura, su valor puede ser una medidad con la unidad correspondiente "10px, 12em"..
  • width = "". Su unidad puede ser cualquier medida con su respectiva unidad de medida.
